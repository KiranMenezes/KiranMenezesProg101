Page Description

When you click the button an event is fired that causes the onclick code to run. Inside the onclick method the code is displaying a list. First the code clears the list in case there is any old material, then the code sorts the elements within an array. After that, the code iterates over the array. In each iteration the code is creating a list item, after the list item is created, it gets added to the list.

The first button diplays numbers, sorted in ascending order. The second button displays names of programming languages, sorted alphebetically. The third button displays names of the programming languages, sorted by popularity.
